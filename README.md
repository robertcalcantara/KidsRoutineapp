# KidsRoutineapp
é um aplicativo mobile desenvolvido com o objetivo de auxiliar pais e responsáveis na organização da rotina diária de crianças. A aplicação permite o cadastro de atividades como tarefas escolares, horários de estudo, momentos de lazer e cuidados pessoais, promovendo uma rotina estruturada e equilibrada.
O sistema possui uma interface intuitiva e amigável, pensada especialmente para facilitar o uso tanto por adultos quanto por crianças. Entre suas principais funcionalidades estão o gerenciamento de perfis infantis, visualização de tarefas diárias, acompanhamento de atividades concluídas e organização por horários.

Além disso, o aplicativo busca incentivar a autonomia e disciplina das crianças por meio de uma abordagem visual simples e interativa, contribuindo para o desenvolvimento de hábitos saudáveis desde cedo.

O projeto foi desenvolvido utilizando Flutter para o front-end, proporcionando uma experiência multiplataforma, com foco em responsividade e usabilidade.
